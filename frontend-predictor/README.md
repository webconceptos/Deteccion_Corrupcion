# Predictor Obras — Frontend (React + Vite + Tailwind)

## Requisitos
- Node.js 18+

## Setup
```bash
npm install
npm run dev
```
La app asume el backend en `http://localhost:8000`. Para cambiarlo, crea `.env`:
```
VITE_API_BASE=http://localhost:8001
```

## Producción
```bash
npm run build
npm run preview
```
